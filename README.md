##  Prologue

> **"영속의 황혼 숲, DuskWood"**
>
> 왕국의 몰락 이후, 숲은 저주받은 나무와 이형의 괴물들로 뒤덮였습니다.
> 기억을 잃은 채 숲의 변두리에서 눈을 뜬 주인공.
>
> 그는 끊임없이 밀려오는 몬스터와 맞서며, 숲에 저주를 내린 두 보스를 처치해 황혼에 감춰진 비밀을 밝혀내야 합니다.
